Institute Management system

Our project have many user role(Admin, Faculty, Teacher, Student, Finance) but don't have  any Registration form, only Login form.
The super admin can create a admin and admin will be create Teacher, Student and Finance Account and they will be login using given username and password from admin. All user role have own profile option and they can Edit there own profile and chance them password.

The Dashboard have separate by user role, like Admin dashboard, Teacher dashboard, Student dashboard, and Faculty dashboard.

Admin dashboard have all section summary, like total students and active student, inactive student, total teacher and active teacher, Inactive teacher, total course and active course Inactive course, total Stuff and active stuff, inactive stuff, course overview, active course and inactive course Attendance list(student attendance details, teacher attendance details, staff attendance details ) with bar, All events details, Notice board, Accounts details(Total earning, total expense, total fees collection, student not paid .......

Teacher dashboard have specific teacher daily class routine, full month attendance summary, Lesson plan....

Student dashboard have student attendance, daily classes list, Schedules, student performance bar, Exam results, leave status, Fees reminder, notice board, syllabus 

Parent dashboard have event list, statistics, leave status and home works and 
Exam results
